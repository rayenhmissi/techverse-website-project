  1. if,else,else if

    2. if statement give a condition to a variable. here an example:
    if (x==1){
    printf("true");
    }
    else if means the opposite of the previous condition. for example:
    else if (x==0){
    printf("false");
    }
    else means that the opposite of if statement.for example the opposite of all the previous if statement is:
    else {
    printf("false");
    }

  2. comparison and logical operations:

    • < :smaller than
    • > :greater than
    • == :equal to
    • <= :smaller or equal than
    • >= :greater or equal than
    • != :not equal
    • && :and
    • || :or
    • % :mod

  3. switch

    4. switch statement is used mostly for simple data types to optimise the code.here is an example:
    switch{
    case 1{
    printf("january");
    }
    case 2{
    printf("february");
    }...
    }
    but it is onlly used for one variable so we can't compare two variables or use '||' or '&&'.
