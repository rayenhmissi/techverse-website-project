introduction

sometimes you need to repeat operations like counting and calculations like sum of sequential numbers.it is a bad idea to write the same code many times.so we use something called loops to repeat code with parametre changes.let's discover them.

table of content

  1. for loop
  2. while loop
  3. do while loop
  1. for loop

    2. for loop has a starting point and a finishing point for example from 1 to 10 so it repeats the operation until passing the finishing point.here an example: let's calculate the sum from 1 to 10 .so we have the counter c from 1 to 10 and we initialise the s variable from 0. here is the code :
    int s=0;//initialising s
    for (int c=0;c <= 10;c++){ //c counter begins from 0 and add 1 to its value every iteration
    s=s+c;//adding c to the sum
    }


  2. while loop

    3. while loop means while the condition is true it repeats the operation until the condition become false.here an example:
    int c=0; //initialising the counter
    int s=0; //initialising the sum s
    while (c < 10){ //while c does not pass the condition
    s+=c; //add c to the sum s
    c++;//add 1 step to the counter
    }


  3. do while loop

    4. do while loop is the same as while loop but with syntax differences. it means do the operation while the condition is true.here an example:
    int c=0,s=0;//initialising s sum and c counter
    do{ s+=c;//add the counter to the sum
    c++;//add 1 step to the counter
    }while (c < 10);//while c does not pass 10
