  1. how to create a function

    2. to create a function in c ,we need to know:

    • data type returned
    • function name
    • parameters needed

    for example if we need to create a function to calculate the sum: first we write int as the data type returned.second,we rename the function and finally we define x and y as parameters like this:
    int sum(x,y){//defining the function
    return x+y;//return the sum of x and y
    }
    sometimes we need to write a function that does not return a simple data type so we use void to do this .here an example :
    void hello(){
    printf("hello");//display the word "hello" in the terminal.
    }
    we can void in the main function too.

  2. how to manage parameters

    3. we can change the value of the parameters like here:
    sum(5,5);
    or define the parameters and get them by user input like here:
    int x=scanf("%d",&x); //input x
    int y=scanf("%d",&y); //input y
    // %d is used for digital number
    // %f is used for float variables

  3. recursion

    4. in normal case,we create the function once and we call it with parameters value changer and cannot call the function with more than the number of defined parameters.but recursion is making the function call itself and change its parameter's value until a stop condition is found.here an example:
    int fact(int x){//calculate the factorial of the parameter x
    if (x <= 1){ //stop function
    return 1;//fact(0) and fact(1) =1
    }
    else{//if x more than 1
    return x*fact(x-1);//multiplying x by factorial of x-1
    } }     here how it work:

    1. fact(5)=5*fact(4)
    2. fact(4)=4*fact(3)
    3. fact(3)=3*fact(2)
    4. fact(2)=2*fact(1)
    5. fact(1)=1
    6. fact(2)=2*1=2
    7. fact(3)=3*2=6
    8. fact(4)=4*6=24
    9. fact(5)=5*24=120

    as you can see the function work from the the parameter value to the stop condition and then start calculation from the value given from the stop condition.
