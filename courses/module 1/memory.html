  1. Random Access Memory (RAM):

    2. RAM is designed to access tasks randomly. It is significantly faster than SSD and HDD drives but typically comes in smaller sizes and at a higher cost. Here’s how it works:
    Imagine you are in a kitchen with many tools to make a cake. You place the tools you need on a table; this table represents the RAM. It allows you to have multiple tools in one place for quick access. If the table doesn’t have enough space, you need to save the other tools in their original place or store them together in another place called virtual memory, like a secondary table. The process of switching between these two types of memory is called swapping. When there is free space on the table, you take the tool you need from the secondary table and place it on the primary table.

  2. ROM (Read Only Memory):

    3. This type of memory, known as Read-Only Memory (ROM), is used to read saved data even when the computer is turned off. It is primarily utilized in embedded devices and external peripherals that require read-only access to user input and cannot modify the stored data. ROM is also essential in computers for booting the system and accessing the BIOS (Basic Input/Output System) 12.

  8. storage drive

    1. Hard disk drive (HDD):

      2. An HDD is composed of mechanical parts that spin when the computer is running. This is why its speed is measured in RPM (Revolutions Per Minute). It is the slowest type of storage but also the most affordable. HDDs are commonly found in older computers. The primary advantage of HDDs is their large storage capacity, allowing you to store a significant amount of data.

    2. solid state drive (ssd):

      3. Unlike HDDs, SSDs are much faster, and their speed ismeasured in MB/s (Megabytes per second) or GB/s (Gigabytes per second) because they use electricity instead of mechanical components.
