  2. gpu technologies

    1. gpu excution units

      2. as the cpu ,the gpu also has cores called excution units that to handle tasks that the gpu can handle like graphics,3d,photos and animation and AI and ML tasks but gpus are very power hungry that why NPU (neural processing unit) exists to handdle AI tasks and allow the gpu to focus on more important tasks.we talk more about NPU in the next lesson.

    2. gpu shaders

      3. Shaders are responsible for instructing the execution units on how to process and render graphics. They utilize shading languages, which vary depending on the graphics API being used.

    3. gpu API

      4. Graphics APIs enable developers to utilize libraries and tools for creating applications that rely on advanced graphics or high-quality games. Popular examples include Metal, DirectX, and Vulkan.

    4. ray tracing technology

      5. Ray tracing is a technology utilized in modern GPUs, such as NVIDIA’s RTX series. It enables the simulation of light tracing objects to create realistic reflections and dynamic lighting effects,resulting in stunning visual quality.

    5. upscaling

      6. upscaling is a way to increase the resolution and the framerate that we will talk about the display lesson but now you should know that what is upscalling .NVIDIA DLSS and AMD FSR and apple metal fx upscaling are using this technology in their GPUs to improve performance.

    6. flops
